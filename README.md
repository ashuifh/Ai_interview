## ğŸš€ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/ashuifh/Ai_interview.git
cd Ai_interview
```

### 2. Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file in the `backend` directory:

```env
GEMINI_API_KEY=your_gemini_api_key
VAPI_PRIVATE_KEY=your_vapi_private_key
OPENAI_API_KEY=your_openai_api_key
FIREBASE_TYPE=service_account
FIREBASE_PROJECT_ID=your_project_id
FIREBASE_PRIVATE_KEY_ID=your_private_key_id
 FIREBASE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\nYOUR_KEY\n-----END PRIVATE KEY-----\n" FIREBASE_CLIENT_EMAIL=your_client_email
 FIREBASE_CLIENT_ID=your_client_id
FIREBASE_AUTH_URI=https://accounts.google.com/o/oauth2/auth
 FIREBASE_TOKEN_URI=https://oauth2.googleapis.com/token
```

Add your Firebase service account key as `firebaseKey.json` in the `backend` directory.

Start the backend server:

```bash
npm start
```

The backend will run on `http://localhost:5001`

### 3. Frontend Setup

```bash
cd vite-project
npm install
```

Start the development server:

```bash
npm run dev
```

The frontend will run on `http://localhost:5173`

## ğŸ”§ Configuration

### Environment Variables

**Backend (.env):**
- `GEMINI_API_KEY` - Google Gemini API key for feedback generation
- `VAPI_PRIVATE_KEY` - VAPI private key for voice AI
- `OPENAI_API_KEY` - OpenAI API key (if needed)

**Firebase Setup:**
1. Create a Firebase project
2. Enable Firestore Database
3. Download service account key as `firebaseKey.json`
4. Place it in the `backend` directory

**VAPI Setup:**
1. Sign up at [VAPI](https://vapi.ai)
2. Create a phone number/assistant
3. Get your API key and assistant ID
4. Update the assistant ID in `App.jsx`

## ğŸ“– Usage

1. **Start the Application**
   - Run backend: `cd backend && npm start`
   - Run frontend: `cd vite-project && npm run dev`

2. **Upload Resume**
   - Click on "Upload Your Resume"
   - Select your resume file (PDF, DOC, or DOCX)

3. **Start Interview**
   - Click "Start Interview"
   - Wait for the interview to begin
   - Answer questions asked by the AI interviewer

4. **Stop Interview**
   - Click "Stop Interview" when done
   - Wait for feedback generation
   - View your detailed feedback report

5. **Review Feedback**
   - Check your overall score
   - Read strengths and weaknesses
   - Review communication and technical analysis
   - See final recommendation
